🧠 About the App

Photo Memory is a brain-training game with quick, challenging tasks that sharpen your visual recall, attention, and cognitive focus. It's designed to help you improve through fun repetition.

Disclaimer: There's no evidence the games will actually improve your memory — this was done purely for fun and entertainment.
